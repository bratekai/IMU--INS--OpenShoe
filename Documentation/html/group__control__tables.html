void precision_gyro_bias_null_calibration ( void  )

Initializes the internal IMU gyro calibration routine.

The internal gyro calibration routine will take the mean gyro value over approx. 15s. During this time the IMU will be off-line (not sending out any interrupts). During this period the IMU should be kept statinarry.
