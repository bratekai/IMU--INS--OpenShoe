  • Compiler: IAR EWAVR32 and GNU GCC for AVR32
  • Supported devices: All AVR32 devices with a USART module can be used.

#define Uc3_trace_init (   debug_addr)    *(U32*)(debug_addr)=debug_addr+4

These macros are used to add traces memory.

First, initialise the trace with Uc3_trace_init(pointer), giving the start address of the memory location where will be stored the trace. Use Uc3_trace(something) to store "something" into the memory. The end of the trace is signaled by the "0xdeadbeef" pattern.
