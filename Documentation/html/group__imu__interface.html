void imu_burst_read ( void  )

Request and reads all sensor output data from IMU.

This function uses the IMU burst read functionallity in which all IMU sensor data (rotation, specific force, temperature, and supply voltage) is output by the IMU after a single request. This way only two clock cycles are required between each read operation. This is faster than only reading out rotation and specific force. The functions first reads in the values in 16-bit intermediate variables and then call the help functions convert_inert_readings() and convert_auxiliary_data() to shift out status bits and scale to SI units.

Parameters:
[out]angular_rates_inVector containing the 3 (x,y,z) angular rates in [rad/sec].
[out]accelerations_inVector containing the 3 (x,y,z) specific force in [m/s^2].
[out]imu_temperaturesVector containing the 3 (x,y,z) temperatur readings in [C].
[out]imu_supply_voltageSupply voltage measurement in [V].

void precision_gyro_bias_null_calibration ( void  )

Initializes the internal IMU gyro calibration routine.

The internal gyro calibration routine will take the mean gyro value over approx. 15s. During this time the IMU will be off-line (not sending out any interrupts). During this period the IMU should be kept statinarry.
