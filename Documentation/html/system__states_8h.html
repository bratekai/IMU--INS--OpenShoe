Detailed Description

Declarations of system states and related information tables.

This header file contains 1) Declarations of system states. 2) Definitions of tables containing information about the system states. The system states can be defined across the program. This file just gather the declarations together such that the related information tables can be filled in. The information tables are used for output functions in external_interfaces.c. This file contians multiple static variables so it should only be included where necessary.
